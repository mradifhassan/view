Learning Linux allows you to know about Computer Archietecture and how Computer actually works behind. Now, I am going to provide some Linux based bash commands that I learnt up to this day-

sudo apt install (package name) and sudo apt update && sudo apt upgrade - In Linux, you will have to use many packages or softwares which are completely free to use as I mentioned above before. To install any packages, use first command prompt and to update or upgrade the system, use second prompt which allows your machine to stay up to date advance because every time many new changes are coming and you have to be adapted, I am not saying you have to do frequently, just ensure when you are doing any new activities or new tasks like installing something or something like that, then do that so. Otherwise, no needed. Actually these are your first and basic commands in Linux. Your command journey will start from here.

neofetch - It allows you to see your hardware and other stuffs but before that you need to install that first!

ls - It shows your folder in your computer and visible tasks. It means Lists.

ls -a - It will show all the files and folders of your machine. It means Lists all and it will lists everything include your hidden files.

ls -A - It will show almost every file and folders of your machine. It means Lists almost all.

cd - It means Change Directory. Which means you can change from your current directory to default directory which is void.

cd Documents- After doing this, it will take you to the Documents directory and then you can do whatever you want. I wrote this as an example. But you can write any valid directory name here instead of Documents. If you don't know the directory names, then just write ls and you will find your directory name there.

vim - Vim is a open source and free text filename or filename or editor for using in Linux. It allows you to take notes (Most likely to as Windows based Notepad) and run programs.

vim (filename.extension) - This is the basic structure of creating a file by using vim.

vim program.c - You will be allowed to create and write C program from vim by using this command.

vim program.cpp - Same as 9, just for C++ code.

gcc/clang - These are the compilers for running C/C++ code. You will need to install it first, but gcc is preinstalled in Linux.

gcc/clang program.c -o program - This is the process for compiling your C program.

./program - This is how you can run your program after compiling.

gcc/clang program.c -o program && ./program - It will allow you perform compiling and running two tasks at once.

time - This stuff allow you to calculate how much time your program has taken if you want to see.

./program && time - This will show you both time and allow you to run the program at once.

gcc/clang program.c -o program && time ./program - You can do those three things at once by adding && (double and symbol) and thats how everything will happen at once && is used for executing multiple tasks at once.

vim text filename or filename or .txt - You can create a txt based file by using this command.

cat text filename or filename or .txt - You will able to view your txt file by running this command and cat is used for viewing a text filename.

lynx/w3m - These are the tools for viewing a website from terminal but before that you need to install them first.

lynx/w3m www.websitename.com - That's how you can view and enter a website.

nmap - This is a powerful tool to view how many devices are connected with your WiFi network and you have to install this stuff first.

nmap -sn 192.168.0.XXX/24 - This is how you can check users and you will see something like (Integer Number) host up what the end which means only that number of devices are using your network. Here -sn stands for scan network.

ip a - You can view your ip address by running this command. Here ip a stands for ipcode> address.

whoami - You can know your username by this.

passwd (your username) - You are allowed to change your password by this and here passwd username means password changing for the user.

ls -la - Which means lists all files and directories include hidden files.

htop - A visual, interactive system monitor which allows you to view CPU, Memory and Processes scenario.

df -h - It allows you to see how much memories your located files have taken, besides it will also show Size, Used, Avail, Use% and Mounted on. It means disk free human readable format like MB, GB etc but not in bytes in the current directory. So, here df shows how much disk space is used and available on file systems and -h displays sizes in a friendly format like MB, GBcode> etc instead of raw bytes.

du -sh * - It shows the total size of each file/foldercode> in the current directory in human-readable format. Here -scode> summarize which shows only the total size per item (doesn't show subfolder sizes separately), -h means human-readable which displays sizes in KB, MB, GB instead of bytes and lastly * means applying to everything in the current directory.

grep -Ri 'text' filename or filename or /path/to/destination - Here grep is a tool which allows you to search a particular word (case insensitive Like 'text' = 'TEXT' = 'TeXt') recursively. Here R stands for recursively and i stands for insensitively. Say for example, I want to search 'sudo' inside a file. The location of this file is /home/alex/Documents. I have to go first in home, then alex and after that Documents. So the command should be like this: grep -Ri 'sudo' /home/alex/Documents and by entering the command, I'll get my desired stuffs.

pdftotext - This package helps us to convert a pdf file to txt file.

pdftotext currentfile.pdf desiredfile.txt - Firstly we need to write the tool name by which we are going to do our desired activity. In this case, we are going to convert pdf to txt format, then we need to write down the pdf file's name and after that our desired txt file's name. And that's how we can convert.

w - The w command in Linux shows who is logged in and what they are doing — including active sessions.

df -aT - Here -T shows file system type (like ext4, tmpfs, etc.)

df -ha - Here -a includes pseudo, duplicate, and inaccessible file systems.

uptime - The uptime command tells you how long the system has been running, how many users are logged in, and the system load average.

who -b - It shows last boot time.

users - It allows you to know about user's name.

mpv - It allows you to view your both mp3 and mp4 file. Of course you need to install this tool first before using it.

mpv file.mp3/mp4 - This is the format how you can view your mp3 or mp4 file.

feh - This tool allows you to view your photos like .jpg/.jpeg/.png or somerhing like that.

feh image.format extenstion - This is the basic syntax of viewing any image through this tool.

mkdir (New folder or Directory) - This command allows you to create a new directory or folder.

rm -r (Directory Name) - This is used for deleting any directory or folder recursively, here -r stands for recursively.

cp (file name) /path/to/destination - This will lead you to copy a file from your default directory to another directory. Say for example your currently now inside Documents directory, now you want to move a .jpg file to Photos directory, then you have to go through cp image.jpg /home/alex/Photos and thats how you can copy a file from your current direxctory to another directory.

cp text.txt another_text.txt - This command will help you to make duplicate file inside your current directory.

cp text.txt /path/to/destination/another_text.txt - If you want to copy a file and take this file by giving a new name to another directory, then you have to use this command.

pwd - This command is used to print writing directory, which means it will print your current directory path actually where you are working right now.

mv my_file.txt (Directory Name) - If you have a file named my_file.txt and want to move it to the Documents directory, you would use that command.
